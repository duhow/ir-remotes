# Orbegozo REH 1060

Brand: Orbegozo  
Model: REH 1060
Type: Radiator / Heater

Battery: CR2032

## Codes

- Uses [F12 Protocol](http://www.hifi-remote.com/johnsfine/DecodeIR.html#F12) code.
- Parameters: Device=3 SubDevice=1
- Requires to repeat the same code at least 4 times to work.
- Ensure gaps are no longer than 7097.

See files in folder to get codes.
